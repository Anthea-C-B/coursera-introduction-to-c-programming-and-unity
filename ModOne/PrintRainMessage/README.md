# PrintRainMessage

Small C# console project that prints a short greeting and a brief message.

## Description

`PrintRainMessage` is a minimal .NET console application that demonstrates a simple program entry point which writes a few lines to the console.

## Prerequisites

- .NET 10 SDK
- (Optional) Microsoft Visual Studio 2026 or later

## Build and run (dotnet CLI)

1. Open a terminal in the project directory `PrintRainMessage`.
2. Restore and build:

```
dotnet build
```

3. Run:

```
dotnet run --project PrintRainMessage
```

## Run in Visual Studio

- Open the solution containing `PrintRainMessage` in Visual Studio and run the project (F5 or Ctrl+F5).

## Project files

- `Program.cs` — main console program that prints output to the console.

## Expected output

The program prints three lines to the console, for example:

```
Hello, World!
Chinese Democracy is done and it's November
Is it raining?
```
